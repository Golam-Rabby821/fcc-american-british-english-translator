How it works

Dictionary-first translations

We only translate terms that exist in our project dictionaries (american-only.js, british-only.js, american-to-british-spelling.js, and titles). Time formats are converted (10:30 ⇄ 10.30) and titles drop/add periods based on locale.

Why this site

Built to test

This UI fronts the same Express API used in the unit and functional tests. It's meant to make verification quick before running the FCC test suite.

API

POST /api/translate

Send { text, locale } and receive the original text plus a highlighted translation. Missing fields, empty text, or an invalid locale return clear error messages.
